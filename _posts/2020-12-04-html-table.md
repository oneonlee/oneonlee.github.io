HTML의 기본 태그 중 테이블 태그를 공부했습니다.

| 태그 | 설명 |
|---|---|
| table | 표 삽입 |
| tr | 표에 행 삽입 |
| th | 표의 제목 셀 생성|
| td | 표의 일반 셀 생성 |
- 테이블 태그

| 태그 | 속성 | 설명 |
|---|---|---|
|table |border | 표의 테두리 두께 지정 |
| <r2> th, td | colspan | 셀의 너비 지정 |
||rowspan |셀의 높이 지정|
